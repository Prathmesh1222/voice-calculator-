Voice Calculator

About this project

What is this?

A next-generation calculator that understands natural language. Speak or type commands to perform arithmetic, calculus, graphing, unit conversions, equation solving, and matrix operations.

Features

  • Voice Input — Speak naturally using your microphone
  • Basic Math — "5 plus 3", "product of 4 and 5"
  • Calculus — "differentiate x squared", "integrate 2x"
  • Graphing — "plot sin x", "graph x squared"
  • Equations — "solve x squared minus 4 equals 0"
  • Unit Conversion — "convert 100 celsius to fahrenheit"
  • Matrix Ops — "determinant of [[1,2],[3,4]]"
  • Image OCR — Upload photos of math problems

Tech Stack

Python
Flask
SymPy
Matplotlib
Web Speech API
Tesseract OCR

Project Info

Built as a final-year exhibition project demonstrating natural language processing, symbolic computation, and modern web technologies.
